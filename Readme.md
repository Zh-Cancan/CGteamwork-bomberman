1、完成游戏逻辑（底层完成，需要绘制哪些基本模型对象，实现控制和实现游戏状态检测）
我的想法是游戏逻辑给出一个大数组，根据这个大数组绘制外显状态

3、需不需要实现碰撞 / 接触，这个东西的具体实现逻辑（和我们的游戏逻辑实现有冲突吗）

4、根据游戏逻辑完成模型导入与绘制循环

5、简单视角移动

6、后续：阴影 / 爆炸效果 / 光源 / ……
